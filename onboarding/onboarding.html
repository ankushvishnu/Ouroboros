⚠ One-time setup required

Ollama blocks browser extension requests by default. Add this to your system environment variables once and you're done.

Windows — add to System Environment Variables Variable name: OLLAMA_ORIGINS Variable value: chrome-extension://*,extension://*

Search Edit the system environment variables → Environment Variables → New → paste the values above → OK. Then restart Ollama.

Mac / Linux — add to ~/.zshrc or ~/.bashrc export OLLAMA_ORIGINS="chrome-extension://*,extension://*"
